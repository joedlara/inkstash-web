## Seller Fee Tiers

### Overview
InkStash offers tiered pricing based on seller type, volume, and verification level. Lower fees for serious sellers encourage commitment and higher inventory.

---

## 1. INDIVIDUAL SELLER TIERS

### Tier 1: Casual Seller (Default)
**Who:** Occasional sellers, testing the platform
**Requirements:** None (just sign up)

| Fee Type | Rate | Details |
|----------|------|---------|
| **Selling Fee** | 12% | Per transaction |
| **Payment Processing** | 2.9% + $0.30 | Standard Stripe fees |
| **Total Fee** | ~15% + $0.30 | On each sale |
| **Payout Schedule** | 7 days | After confirmed delivery |
| **Listing Limit** | 50 active | Max simultaneous listings |

**Example:** $100 sale
- Selling fee: $12
- Payment processing: $3.20
- **Seller receives: $84.80** (payout in 7 days)

---

### Tier 2: Active Seller
**Who:** Regular sellers with consistent activity
**Requirements:**
- 10+ completed sales, OR
- $1,000+ total sales, OR
- 50+ active listings

| Fee Type | Rate | Details |
|----------|------|---------|
| **Selling Fee** | 10% | Per transaction |
| **Payment Processing** | 2.9% + $0.30 | Standard Stripe fees |
| **Total Fee** | ~13% + $0.30 | On each sale |
| **Payout Schedule** | 5 days | After confirmed delivery |
| **Listing Limit** | 200 active | Max simultaneous listings |

**Benefits:**
- Lower fees (2% reduction)
- Faster payouts (5 days vs 7)
- Priority customer support
- "Active Seller" badge on profile

**Example:** $100 sale
- Selling fee: $10
- Payment processing: $3.20
- **Seller receives: $86.80** (payout in 5 days)

---

### Tier 3: Power Seller
**Who:** High-volume sellers, near-professional
**Requirements:**
- 50+ completed sales, OR
- $5,000+ total sales, OR
- 4.5+ star rating with 25+ reviews

| Fee Type | Rate | Details |
|----------|------|---------|
| **Selling Fee** | 8% | Per transaction |
| **Payment Processing** | 2.9% + $0.30 | Standard Stripe fees |
| **Total Fee** | ~11% + $0.30 | On each sale |
| **Payout Schedule** | 3 days | After confirmed delivery |
| **Listing Limit** | 500 active | Max simultaneous listings |

**Benefits:**
- Lowest individual fees (8% selling fee)
- Fast payouts (3 days)
- Priority support + dedicated account manager
- "Power Seller" badge with star icon
- Featured in seller directories
- Access to bulk listing tools

**Example:** $100 sale
- Selling fee: $8
- Payment processing: $3.20
- **Seller receives: $88.80** (payout in 3 days)

## 3. LIVE STREAM SELLER TIERS

Live streaming sellers pay the same base fees **plus** a small platform fee for streaming infrastructure.

### Standard Streamer
**Requirements:** Any verified seller

| Fee Type | Rate | Details |
|----------|------|---------|
| **Base Selling Fee** | Per tier | Same as seller tier (8-12%) |
| **Streaming Platform Fee** | 2% | For video infrastructure |
| **Payment Processing** | 2.9% + $0.30 | Standard Stripe fees |
| **Total Fee** | ~13-17% + $0.30 | Depending on seller tier |

**Example:** Casual Seller streaming, $100 live auction sale
- Selling fee: $12
- Streaming fee: $2
- Payment processing: $3.20
- **Seller receives: $82.80**

## Alpha Tester Incentives

### Strategy
Attract early adopters with aggressive pricing, then grandfather them into lifetime discounts.

---

### Alpha Tester Program (First 100 Sellers)

**Duration:** First 6 months after sign-up or until 100 sellers reached

| Benefit | Details |
|---------|---------|
| **Selling Fee** | **5% flat** (50-60% discount) |
| **Payment Processing** | 2.9% + $0.30 (standard Stripe, no markup) |
| **Total Fee** | **~8% + $0.30** (vs 15% for casual sellers) |
| **Payout Schedule** | 3 days (vs 7 days standard) |
| **Listing Limit** | 500 active (vs 50 standard) |
| **Streaming Fee** | **Waived** (free streaming for 6 months) |
| **Lifetime Badge** | "Alpha Tester" badge on profile |

**After 6 months:**
- Transition to **7% selling fee** (permanent grandfather rate)
- Still better than standard rates
- "OG Seller" badge replaces "Alpha Tester"

**Example:** $100 sale during alpha
- Selling fee: $5 (60% off!)
- Payment processing: $3.20
- **Seller receives: $91.80** (payout in 3 days)

## Implementation Guide

### Database Schema

Add to `users` or `seller_profiles` table:

```sql
-- Add to users table
ALTER TABLE users ADD COLUMN seller_tier VARCHAR(50) DEFAULT 'casual';
ALTER TABLE users ADD COLUMN seller_tier_updated_at TIMESTAMP;
ALTER TABLE users ADD COLUMN is_alpha_tester BOOLEAN DEFAULT FALSE;
ALTER TABLE users ADD COLUMN is_beta_tester BOOLEAN DEFAULT FALSE;
ALTER TABLE users ADD COLUMN grandfathered_fee_rate DECIMAL(5,2); -- Permanent rate for alpha/beta

-- Create seller tier history table
CREATE TABLE seller_tier_history (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  previous_tier VARCHAR(50),
  new_tier VARCHAR(50),
  changed_at TIMESTAMP DEFAULT NOW(),
  reason TEXT -- 'automatic', 'manual', 'alpha_upgrade', etc.
);

-- Create seller statistics table (for tier calculation)
CREATE TABLE seller_statistics (
  user_id UUID PRIMARY KEY REFERENCES users(id),
  total_sales_count INTEGER DEFAULT 0,
  total_sales_value DECIMAL(10,2) DEFAULT 0,
  active_listings_count INTEGER DEFAULT 0,
  average_rating DECIMAL(3,2),
  total_reviews_count INTEGER DEFAULT 0,
  total_streams_count INTEGER DEFAULT 0,
  total_stream_sales DECIMAL(10,2) DEFAULT 0,
  follower_count INTEGER DEFAULT 0,
  last_sale_at TIMESTAMP,
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Function to calculate seller tier
CREATE OR REPLACE FUNCTION calculate_seller_tier(user_id_param UUID)
RETURNS VARCHAR(50) AS $$
DECLARE
  stats RECORD;
  new_tier VARCHAR(50);
BEGIN
  SELECT * INTO stats FROM seller_statistics WHERE user_id = user_id_param;

  -- Check for Business verification
  IF EXISTS (SELECT 1 FROM seller_onboarding WHERE user_id = user_id_param AND business_verified = TRUE) THEN
    RETURN 'business';
  END IF;

  -- Check for Power Seller
  IF stats.total_sales_count >= 50 OR stats.total_sales_value >= 5000 OR
     (stats.average_rating >= 4.5 AND stats.total_reviews_count >= 25) THEN
    RETURN 'power';
  END IF;

  -- Check for Active Seller
  IF stats.total_sales_count >= 10 OR stats.total_sales_value >= 1000 OR stats.active_listings_count >= 50 THEN
    RETURN 'active';
  END IF;

  -- Default to Casual
  RETURN 'casual';
END;
$$ LANGUAGE plpgsql;

-- Trigger to auto-update tier after each sale
CREATE OR REPLACE FUNCTION update_seller_tier_on_sale()
RETURNS TRIGGER AS $$
BEGIN
  UPDATE users
  SET seller_tier = calculate_seller_tier(NEW.seller_id),
      seller_tier_updated_at = NOW()
  WHERE id = NEW.seller_id;

  RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER trigger_update_seller_tier_on_sale
AFTER INSERT ON orders
FOR EACH ROW
EXECUTE FUNCTION update_seller_tier_on_sale();
```

### Fee Calculation Function

```typescript
// src/utils/feeCalculator.ts

interface FeeBreakdown {
  sellingFee: number;
  sellingFeeRate: number;
  paymentProcessingFee: number;
  streamingFee?: number;
  totalFees: number;
  sellerReceives: number;
  payoutDays: number;
}

interface User {
  seller_tier: 'casual' | 'active' | 'power' | 'business' | 'enterprise';
  is_alpha_tester: boolean;
  is_beta_tester: boolean;
  grandfathered_fee_rate?: number;
  total_streams_count?: number;
  follower_count?: number;
}

export function calculateFees(
  saleAmount: number,
  user: User,
  isLiveStream: boolean = false
): FeeBreakdown {
  // Determine base selling fee
  let sellingFeeRate: number;

  // Check for grandfathered rates (alpha/beta)
  if (user.grandfathered_fee_rate) {
    sellingFeeRate = user.grandfathered_fee_rate;
  }
  // Alpha testers get 5%
  else if (user.is_alpha_tester) {
    sellingFeeRate = 0.05;
  }
  // Beta testers get 7%
  else if (user.is_beta_tester) {
    sellingFeeRate = 0.07;
  }
  // Standard tier-based rates
  else {
    switch (user.seller_tier) {
      case 'enterprise':
        sellingFeeRate = 0.055; // 5.5% average (negotiable 5-6%)
        break;
      case 'business':
        sellingFeeRate = 0.07;
        break;
      case 'power':
        sellingFeeRate = 0.08;
        break;
      case 'active':
        sellingFeeRate = 0.10;
        break;
      case 'casual':
      default:
        sellingFeeRate = 0.12;
    }
  }

  const sellingFee = saleAmount * sellingFeeRate;

  // Payment processing (Stripe standard)
  const paymentProcessingFee = saleAmount * 0.029 + 0.30;

  // Streaming fee (if applicable)
  let streamingFee = 0;
  if (isLiveStream) {
    // Premium streamers get 1%, standard get 2%
    const isPremiumStreamer =
      (user.total_streams_count && user.total_streams_count >= 10) ||
      (user.follower_count && user.follower_count >= 500);

    const streamingFeeRate = isPremiumStreamer ? 0.01 : 0.02;

    // Alpha testers get streaming waived
    if (!user.is_alpha_tester) {
      streamingFee = saleAmount * streamingFeeRate;
    }
  }

  const totalFees = sellingFee + paymentProcessingFee + streamingFee;
  const sellerReceives = saleAmount - totalFees;

  // Determine payout schedule
  let payoutDays: number;
  if (user.is_alpha_tester || user.seller_tier === 'power') {
    payoutDays = 3;
  } else if (user.is_beta_tester || user.seller_tier === 'active') {
    payoutDays = 5;
  } else if (user.seller_tier === 'business') {
    payoutDays = 2;
  } else if (user.seller_tier === 'enterprise') {
    payoutDays = 1; // Next-day or instant
  } else {
    payoutDays = 7; // Casual sellers
  }

  return {
    sellingFee: Math.round(sellingFee * 100) / 100,
    sellingFeeRate,
    paymentProcessingFee: Math.round(paymentProcessingFee * 100) / 100,
    streamingFee: streamingFee > 0 ? Math.round(streamingFee * 100) / 100 : undefined,
    totalFees: Math.round(totalFees * 100) / 100,
    sellerReceives: Math.round(sellerReceives * 100) / 100,
    payoutDays
  };
}

// Example usage:
const user = {
  seller_tier: 'active',
  is_alpha_tester: false,
  is_beta_tester: false,
  total_streams_count: 5,
  follower_count: 200
};

const fees = calculateFees(100, user, false);
console.log(fees);
// {
//   sellingFee: 10.00,
//   sellingFeeRate: 0.10,
//   paymentProcessingFee: 3.20,
//   totalFees: 13.20,
//   sellerReceives: 86.80,
//   payoutDays: 5
// }
```
